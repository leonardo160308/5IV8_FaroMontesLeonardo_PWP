09Node

En esta práctica se creó un servidor básico utilizando Node.js. El programa usa el módulo http para permitir la comunicación entre el cliente y el servidor. Al ejecutar el código y acceder desde el navegador a la dirección http://localhost:3000, el servidor responde enviando una página con el mensaje “Hola Mundo desde Node.js” junto con otros textos. Además, se muestra en la consola un mensaje que confirma que el servidor está funcionando correctamente. Esta práctica sirve para comprender cómo Node.js maneja las solicitudes (request) y respuestas (response) dentro del modelo cliente-servidor.
10Postman

En esta práctica se desarrolló un servidor utilizando el framework Express de Node.js. El programa crea un servidor que escucha en el puerto 3000 y utiliza un archivo externo llamado mirrow para manejar las peticiones. Se configuraron varias rutas (GET, POST, PUT, PATCH, DELETE y HEAD) que permiten interactuar con el servidor desde diferentes métodos HTTP. También se agregó el middleware express.json() para procesar datos en formato JSON enviados por el cliente. Esta práctica tiene como propósito comprender el uso básico de Express y cómo definir diferentes tipos de peticiones dentro de una aplicación servidor.
